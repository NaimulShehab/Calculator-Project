# Calculator Web App

A simple calculator web application built using **HTML, CSS, and JavaScript**.

## About the Project
- The **HTML structure** and **JavaScript logic** were implemented by me.
- **AI assistance** was used for the CSS design to enhance the UI and overall styling.

## Features
- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear (**C**) button to reset the display
- Error handling for invalid expressions
- Responsive and modern user interface

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## Purpose
This project was created to practice:
- DOM manipulation
- Event handling
- Basic JavaScript logic  

It also helped me understand how **HTML, CSS, and JavaScript** work together in a real-world project.
